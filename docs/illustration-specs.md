# EndoCyclic — Scientific Illustration Prompt Specs

Nature-Reviews-grade illustration prompts you can paste straight into **ChatGPT's image generator** (GPT-4o / DALL·E) to enrich the website. Every prompt is grounded in `truth.md` and uses the site's **Calm Clinical** palette, so the results drop in cohesively.

## How to use each spec
1. Open ChatGPT (image generation), paste the **Prompt** block verbatim, and add the aspect ratio in words (e.g. "make it a wide 16:9 landscape").
2. Expect **2–3 iterations** — use the *Avoid* and *Usage* notes to steer follow-ups ("no text anywhere", "keep the rose region small and calm", "lighter, more off-white background").
3. Export the result, convert to **.webp** (keep the long edge ~2000px), and save it at the **Suggested file** path.
4. Tell me the file is in place and I'll wire it into the page (hero background, section image, etc.).

## Shared conventions (already baked into every prompt)

**Locked anchor** (the first line of every prompt — keeps the whole set consistent):
> Soft watercolor scientific illustration in the editorial style of a Nature Reviews journal — muted, desaturated, anatomically grounded, calm and restrained, soft diffused edges, generous negative space, publication quality, no text, no labels, no neon, no glossy 3D, no harsh gradients.

**Calm Clinical palette** (max 5 color areas per image; colorblind-safe; never red/green):

| Meaning | Color | Hex |
|---|---|---|
| Background (or transparent) | soft off-white | `#FBFAF8` |
| **The peptide / healthy / active** (the hero, most saturated) | clinical teal | `#4A9B8E` / `#2F6E62` |
| Anatomy outlines / structure | muted plum-ink | `#2E263A` |
| Disease / lesion / acidic zone (sparing) | dusty rose | `#C98B84` / `#FBF4F3` |
| Neutral / inert / context | soft gray + tints | `#B7B2BB` `#F4F1F8` `#EDF5F2` `#F6F3EE` |

**Honest caveat.** AI image gen excels at *atmospheric, conceptual* scientific art (heroes, molecular impressions, tissue scenes) — that's what these specs target. It is *not* reliable for precise labeled molecular diagrams or in-image text, so every prompt says "no text"; on-page labels stay in HTML, and the exact labeled diagrams remain the built-in SVG figures.
