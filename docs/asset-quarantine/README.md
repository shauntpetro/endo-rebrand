# Asset quarantine

These files were removed from the public delivery surface because their visual
provenance or implied claim is not suitable for a pre-approval clinical-stage
website:

- `standard-mri.webp` and `FemLUNA-enhanced.webp` resemble comparative clinical
  scan output and include unapproved measurement-style annotations.
- `recognition-perfect10.webp` is a generic campus-style image that does not
  document the NIH award it was associated with.

They are retained here for recovery and provenance review. Do not restore them
to `public/` without an approved source record and usage context.
